# Fichiers du clavier

Ce dossier contient les fichiers nécessaires à la fabrication et à l'impression du clavier braille.

## Contenu

- **Braille_Keyboard.pdf** : version prête à imprimer.
- **Braille_Keyboard.svg** : fichier vectoriel modifiable pour la découpe ou la personnalisation.

Ces fichiers peuvent être utilisés pour reproduire le clavier développé dans ce projet.
